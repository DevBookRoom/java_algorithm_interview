## ✅ 과제 제출 방식 
1. 기본 작업환경은 본인 이름의 브랜치에서 작업합니다.
2. 주차별 폴더에 자기 이름의 폴더를 생성한 후 푼 내용을 커밋합니다.
3. 리뷰어를 지정한 후 목요일까지 푸쉬를 합니다.
4. 리뷰어분은 코드를 스터디 전까지 확인 후 리뷰합니다.
5. 방장은 발표가 끝난 후 각 PR들을 머지합니다.

2. 발표가 끝난 후 질의응답 시간을 가집니다.
